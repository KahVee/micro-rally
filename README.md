# Micro machines

## Required libraries

**Box-2D:**  
Install on Linux:
```
sudo apt-get install libbox2d-dev
```

**SFML**  
Install on Linux:
```
sudo apt-get install libsfml-dev
```

## Required build tools
**Gcc**
```
sudo apt-get install build-essential manpages-dev
```

**CMake**
```
sudo apt-get install cmake
```
